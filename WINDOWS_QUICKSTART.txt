## Getting Started (3 Steps)

### Step 1: Choose Your Mode
**If your package includes `SAP_SOAP_Client.exe`:**
- No Node.js installation required

**If your package does NOT include the EXE:**
1. Visit https://nodejs.org/
2. Download the LTS version (18.x or 20.x)
3. Run the installer and follow the prompts
4. After installation, restart your computer

### Step 2: Extract the Package
1. Right-click the ZIP file
2. Select "Extract All..."
3. Choose a location (e.g., C:\Programs or Desktop)
4. Wait for extraction to complete

### Step 3: Run the Application
Choose ONE of these options:

#### Option A: PowerShell (Recommended for Windows 10/11)
- Double-click `start.ps1`
- If prompted about execution policy, click "Run Anyway"
- Browser will open automatically

#### Option B: Batch File
- Double-click `start.bat`
- Wait for "Dependencies installed" message
- Browser will open automatically

#### Option C: Manual (Command Prompt)
```
1. Open Command Prompt
2. Navigate to the folder with "cd C:\path\to\SAP_SOAP_Client"
3. Run: npm install
4. Run: npm start
5. Open http://localhost:3000 in your browser
```

## Configuration

### SAP System Settings
1. Enter your SAP system URL
2. Provide your SAP username and password
3. Click "Save Credentials"

### PC Miler API (for geocoding)
1. Get your API key from PC Miler
2. In Configuration section, fill in:
   - **PC Miler API Key**: Your API key
   - **PC Miler API Endpoint**: Default is pre-filled
3. Click "Save PC Miler Config"

## Troubleshooting

### Issue: "Node.js is not installed"
**Fix**: Use the standalone package with `SAP_SOAP_Client.exe`, or install Node.js and restart your computer

### Issue: PowerShell won't run the script
**Fix**: 
1. Open PowerShell as Administrator
2. Run: `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`
3. Type Y and press Enter
4. Try `start.ps1` again

### Issue: "port is already in use"
**Fix**: The script will automatically use port 3001, 3002, etc. if 3000 is taken

### Issue: Browser doesn't open
**Fix**: Manually visit http://localhost:3000

### Issue: npm not found
**Fix**: Restart your computer after installing Node.js

### Issue: Dependencies install fails
**Fix**: 
1. Delete the `node_modules` folder
2. Run `start.bat` or `start.ps1` again
3. It will reinstall dependencies
